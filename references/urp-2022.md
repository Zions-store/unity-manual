# URP in Unity 2022.3 LTS

## Pipeline Version
- URP 14.x
- RenderGraph API: experimental/opt-in via Project Settings
- ShaderGraph 14.x

## Key Features Available
- Forward+ rendering path (for many dynamic lights)
- Decal system (DBuffer)
- Screen Space Ambient Occlusion (SSAO)
- Reflection Probes (baked and realtime)
- Camera Stacking: Base + Overlay camera system
- 2D Renderer with 2D lights and shadows
- Post-processing via Volume framework (Bloom, Vignette, Color Adjustments, etc.)
- SRP Batcher for draw call optimization
- Custom Renderer Features via ScriptableRendererFeature

## Key Limitations vs Unity 6
- No GPU Resident Drawer
- No Probe Volumes (Adaptive Probe Volumes are Unity 6+)
- Shader stripping less aggressive
- No Render Graph by default (must enable manually)
- Limited cross-platform performance compared to Unity 6
